# ☁️ Cloud Intrusion Detection & Video Analytics

An advanced computer vision project focused on processing video streams and identifying unauthorized intrusions using deep learning architectures. It is structured for rapid prototyping and cloud deployment.

## 🌟 Key Features
* **Stream Processing:** Includes specialized Jupyter notebooks (`live-stream(1).ipynb`) for pulling, decoding, and analyzing continuous video feeds (RTSP/Webcam/Cloud URLs).
* **Algorithm Prototyping:** The core notebook (`notebook83fc804cb8.ipynb`) contains the foundational algorithms for background subtraction, object tracking, and deep-learning-based intrusion verification.
* **Modular Pipeline:** Easy to adapt for edge devices or deploy onto cloud VMs for scalable video analytics.

## 🛠️ Tech Stack
* Python, Jupyter Notebook
* OpenCV (Image Processing)
* Deep Learning Frameworks (TensorFlow/PyTorch)

## 🚀 Usage
Open the `.ipynb` files in Jupyter or Google Colab. The notebooks are heavily commented and structure the data ingestion, model inference, and output visualization steps.
